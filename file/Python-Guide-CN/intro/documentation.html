pydoc¶

pydoc 是一个在您安装Python时跟着安装的工具。它允许您在shell中快速检索和查找文档。比如，如果您需要对 time 模块的进行快速回顾，查看文档就是像下面这么简单：

$ pydoc time

上面的命令和打开Python REPL然后运行下面指令是基本等价的：

>>> help(time)

这份较为主观的指南旨在为Python初学者和专家提供一个关于Python安装、配置、和日常使用的最佳实践手册。
