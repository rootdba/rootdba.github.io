Mongo命令连接远程服务器的MongoDB数据库 - MongoDB

日期：2017.01.17

MongoDB连接远程服务器的命令格式如下:

mongo 远程主机ip或DNS:MongoDB端口号/数据库名 -u user -p password

MongoDB连接远程服务器的命令示例代码:

//使用默认端口连接MongoDB

mongo 192.168.1.100
MongoDB shell version: 2.4.8
connecting to: 192.168.1.100/test

//连接MongoDB并指定端口
mongo 192.168.1.100:27017

//连接到指定的MongoDB数据库
mongo 192.168.1.100:27017/test

//指定用户名和密码连接到指定的MongoDB数据库
mongo 192.168.1.200:27017/test -u user -p password

注意如果连接到远程服务器，需要新建入站规则，让外面可以连接到远程服务器的指定端口

iptables -A INPUT -s 192.168.121.160 -p tcp --destination-port 27019 -m state --state NEW,ESTABLISHED -j ACCEPT
iptables -A OUTPUT -d 192.168.121.160 -p tcp --source-port 27019 -m state --state ESTABLISHED -j ACCEPT

创建iptables允许27019端口的出入站；
