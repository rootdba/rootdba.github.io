Nginx以普通用户运行 - Linux

日期：2017.01.12

root账号执行：

useradd nginx
chown -R nginx.nginx /usr/local/nginx/
setcap CAP_NET_BIND_SERVICE=+ep /usr/local/nginx/sbin/nginx
killall -9 nginx
su - nginx
/usr/local/nginx/sbin/nginx

查看进程，nginx主进程及子进程都是以nginx用户在运行了；
