Win11部署MySQL Community Server 8.0.31 绿色免安装

日期: 2022-10-12 分类: MySQL
全文共939字，阅读完毕需 2 分钟
Tags: MySQL 安装 Win11 绿色 部署
1、先在MySQL官方下载安装包：
1--MySQL官方下载地址:
2https://dev.mysql.com/downloads/mysql/
3--直接下载地址：
4https://cdn.mysql.com//Downloads/MySQL-8.0/mysql-8.0.31-winx64.zip
2、解压并设置环境变量：

将mysql-8.0.31-winx64.zip解压到C盘根目录下，解压完成后会有C:\mysql-8.0.31-winx64目录。 添加环境变量： 电脑->右键 属性->高级系统设置->环境变量->系统变量->PATH，在PATH里添加入mysql安装路径下的bin的路径—例如我的是：C:\mysql-8.0.31-winx64\bin\

3、创建my.cnf配置文件：
 1--在C:\mysql-8.0.31-winx64目录下创建my.cnf文件，内容如下：
 2[mysql]
 3default-character-set=utf8mb4
 4[mysqld]
 5--设置服务端口为3306
 6port=3306
 7--设置mysql的安装目录，注意目录需要使用\\连接
 8basedir=C:\\mysql-8.0.31-winx64
 9--设置mysql数据库的数据的存放目录，注意目录需要使用\\连接
10datadir=C:\\mysql-8.0.31-winx64\\data
11--允许最大连接数
12max_connections=200
13--允许连接失败的次数。这是为了防止有人从该主机试图攻击数据库系统
14max_connect_errors=100     
15character-set-server=utf8mb4
16--创建新表时将使用的默认存储引擎
17default-storage-engine=INNODB
18--默认使用“mysql_native_password”插件认证
19default_authentication_plugin=mysql_native_password
20log-bin = C:\\mysql-8.0.31-winx64\\binlog\\binlog
21sync_binlog = 1
22binlog_cache_size = 4M
23max_binlog_cache_size = 2G
24max_binlog_size = 1G
25slow_query_log_file = C:\\mysql-8.0.31-winx64\\slow.log
26log-error = C:\\mysql-8.0.31-winx64\\error.log
27
28[client]
29--设置mysql客户端连接服务端时默认使用的端口
30port=3306
31default-character-set=utf8mb4
4、创建data目录、安装服务及初始化：
 1--在C:\mysql-8.0.31-winx64下创建data和binglog目录
 2--以管理员账号运行CMD，执行初始化引导：
 3PS C:\Users\ice> mysqld --initialize --console
 42022-10-12T04:26:16.382015Z 0 [Warning] [MY-010918] [Server] 'default_authentication_plugin' is deprecated and will be removed in a future release. Please use authentication_policy instead.
 52022-10-12T04:26:16.382036Z 0 [System] [MY-013169] [Server] C:\mysql-8.0.31-winx64\bin\mysqld.exe (mysqld 8.0.31) initializing of server in progress as process 9528
 62022-10-12T04:26:16.404446Z 1 [System] [MY-013576] [InnoDB] InnoDB initialization has started.
 72022-10-12T04:26:16.641875Z 1 [System] [MY-013577] [InnoDB] InnoDB initialization has ended.
 82022-10-12T04:26:17.411693Z 6 [Note] [MY-010454] [Server] A temporary password is generated for root@localhost: ,KOtD%adf2zD
 9
10--此处会显示mysql的初始密码如上所示。
11
12--安装服务：
13C:\Windows\System32> mysqld --install mysqld
14Service successfully installed.
15
16--启动服务：
17C:\Windows\System32>net start mysqld
18mysqld 服务正在启动 .
19mysqld 服务已经启动成功。
5、修改默认密码：
1mysql -uroot -p
2--输入密码即可登录
3--修改密码：
4ALTER USER root@'localhost' IDENTIFIED WITH mysql_native_password BY 'YOUPASSWORD';
5flush privileges;
6、进行初始化设置：
 1--在cmd命令行中运行
 2mysql_secure_installation
 3--回车后根据提示设置，最后重启mysql服务
 4C:\Windows\System32>net stop mysqld
 5mysqld 服务正在停止..
 6mysqld 服务已成功停止。
 7
 8C:\Windows\System32>net start mysqld
 9mysqld 服务正在启动 .
10mysqld 服务无法启动。

至此MySQL8.0.31已部署完成，可以根据实际需求配置修改my.cnf文件。
