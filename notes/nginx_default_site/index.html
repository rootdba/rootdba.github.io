Nginx禁止ip直接访问或任意域名访问

日期: 2022-10-01 分类: Nginx
全文共97字，阅读完毕需 1 分钟
Tags: Nginx 禁止 访问 域名
禁止未绑定的域名或任意域名直接访问：
1server
2{
3	listen 80 default_server;
4	server_name _; # 默认server
5	return 500;
6}
对任意域名访问做301跳转：
1server {
2	listen 80 default_server;
3	server_name _;
4  	return       301 https://www.test.com$request_uri;
5}

以上设置后使用IP不能直接访问服务器，需要绑定域名后访问。
