发行说明¶

Django 官方版本的发行说明。每个版本的发行说明都会告诉你每个版本的新内容，也会描述该版本中任何向后不兼容的改动。

对于那些 升级到新版本的 Django，你需要检查所有向后不兼容的变化和 废弃的功能，从你当前的 Django 版本之后的每一个 “最终” 版本，直到并包括新版本。

最终版本¶

下面是通过 Django 3.2 及其补丁发布的发行说明。新版本的文档包含了以后版本的发行说明。

此外，还维护着 已披露的安全问题档案。
